# Научиться учиться
-------------------
## Описание
Данный проект посвящен введению в курс Web-разработчика. В ходе работы мы сверстали сайт, адаптирующийся под размер окна браузера. Используются следующие технологии:  
1. Флексбокс-вёрстка.
2. Изображения
3. Анимация и трансформации
4. Позиционирование
5. Видео
